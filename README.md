# IPL Business Intelligence & Advanced Cricket Analytics Project

##  Project Overview

This project is an end-to-end Data Analytics and Business Intelligence solution built using IPL historical cricket data.

The project covers:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Advanced SQL Analytics
- Business Insights
- Dashboard Preparation

---

##  Objectives

- Analyze IPL team and player performance
- Perform advanced cricket analytics
- Generate business insights using SQL
- Create dashboard-ready datasets
- Build a complete BI workflow

---

##  Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- MySQL
- Jupyter Notebook
- Power BI (Upcoming)

---

##  Datasets Used

### Matches Dataset
Contains:
- Match details
- Teams
- Toss results
- Venue information
- Match winners

### Deliveries Dataset
Contains:
- Ball-by-ball records
- Batting performance
- Bowling performance
- Extras and wickets

---

##  Key Analysis Performed

### Python EDA
- Team win analysis
- Venue analysis
- Toss impact
- Top batsmen and bowlers
- Powerplay and death-over analysis

### SQL Analytics
- Window Functions
- CTEs
- Ranking analysis
- Venue dominance
- Strike rate analysis
- Batter vs Bowler analysis

---

##  Sample Business Insights

- Some venues favor chasing teams
- Aggressive powerplay batting increases win probability
- Death-over specialists strongly influence outcomes
- Toss decisions impact match strategy
